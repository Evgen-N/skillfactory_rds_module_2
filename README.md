# skillfactory_rds_module_2
EDA - разведывательный анализ данных 
Требуется отобрать данные из условий жизни учащихся в возрасте от 15 до 22 лет, влияющие на их успеваемость 
по математике
Размер датафрейма 30х395
3 количественных признака - age, absence и score(это целевой признак)
10 порядковых признаков
17 категориальных признаков
Сначала меняем названия столбцов
Затем удаляем один порядковый столбец, на 100% коррелирующий с другим
Потом определяем необходимые для обработки признаков функции
Обрабатываем все признаки по-очереди согласно их типу, подсчитываем уникальные и пустые значения, 
рисуем графики, убираем выбросы
Проводим корреляционный анализ для количественных признаков
Делаем тест Тьюринга на гипотезу о нулевой разнице параметров распределений итоговой оценки(score)
по парам значений из категориальных или порядковых признаков на уровне доверия 95% с учетом поправки Бонферрони
Отбираем в итоге 8 столбцов для дальнейшего анализа целевой переменной
Делаем выводы из проведенного исследования
